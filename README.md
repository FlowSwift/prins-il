# Prins - Israel

## Development

Enable virtual environment:

```sh
python3 -m venv venv
. venv/bin/activate
pip install -r requirements.txt
export FLASK_APP=application
export FLASK_ENV=development # enable autoreload
```

## Running

```sh
flask run # bash
```
 
Visit http://127.0.0.1:5000/
